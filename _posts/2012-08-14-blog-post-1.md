Architecture is the most important part of a project. I have some insights on how to construct an elligent project:
- pass file directory/path instead of generating one inside the class/function
- make the function as small as possible so that it can be reused by many higher-level functions
- consider the actual using scene/workflow before making any meaningless optimization
